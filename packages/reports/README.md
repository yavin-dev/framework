# navi-reports

Analytics reporting library for navi

## Compatibility

- Ember.js v3.16 or above
- Ember CLI v2.13 or above
- Node.js v10 or above

## Installation

```
ember install navi-reports
```

## Usage

- Including CSS styling:
  - Add `@import "navi-reports";` into your `app.scss`

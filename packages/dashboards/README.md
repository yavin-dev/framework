# navi-dashboards

Analytics dashboard library for navi

- Install Addon

  - `ember install navi-dashboards`

- Including CSS styling:
  - Add `@import "navi-dashboards";` into your `app.less`

## Requirements

- Ember.js v3.12 or above
- Ember CLI v2.13 or above
- Node.js v10 or above
